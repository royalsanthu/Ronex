Fully Customizable Parameters

Fine-tune Ronex directly from the MT5 inputs tab to match your specific risk profile and portfolio structure.

1. Structural Entry Settings

  • EMA 200 Period: Defines the macro trend direction baseline.
  • EMA 20 Period: Defines the minor pullback structure zone.
  • Risk-to-Reward Ratio: Custom targets (e.g., 2.0 = 1:2 R:R).

2. Risk & Account Limits

  • Fixed Lot Size: Sets the base trade entry volume.
  • Max Lot Limit: Caps maximum trade volume for safety.
  • Daily Profit Target Cap ($): Daily yield ceiling to halt operations.
  • Daily Loss Limit Cap ($): Absolute maximum daily loss boundary.

3. Defensive Smart Grid

  • Max Grid Levels: Hard recovery cap (default 3 levels).
  • Grid Spacing (Pips): PIP distance spacing between recovery orders.
  • Grid Multiplier: Multiplier volume (1.0 = same size recovery).
  • Basket Net Target (Pips): Profit target for average break-even exit.

4. Trend Strength Filter (ADX)

  • ADX Period: Average period smoothing for trend tracking.
  • Minimum ADX Level: Entry threshold level (default 22.0).
